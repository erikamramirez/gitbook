# Página Web con GitHub

## <mark style="color:purple;">¿Cómo diseñar una página web gratuita con GitHub?</mark>

GitHub es una herramienta altamente recomendada para la creación de nuestro portafolio o página web de forma gratuita. A continuación, te invito a seguir los siguientes pasos para comenzar este proceso:

### <mark style="color:purple;">Creación de la cuenta de usuario.</mark>

* Abre tu navegador web y ve a la página principal de GitHub en el siguiente link.

{% embed url="https://github.com/" %}
liko para GitHub.
{% endembed %}

* Localizamos ¨Sign up¨ para registrarnos a la plataforma.

<figure><img src=".gitbook/assets/imagen_2023-09-19_000940361.png" alt=""><figcaption><p>En el margen superior derecho encontramos Sign Up, hacemos click para registrarnos en la plataforma.</p></figcaption></figure>

* Ahora debemos ingresar nuestros datos personales.

<figure><img src=".gitbook/assets/imagen_2023-09-19_001153926.png" alt=""><figcaption><p>Ingresar email, por ejemplo tu cuenta de Gmail.</p></figcaption></figure>

<figure><img src=".gitbook/assets/imagen_2023-09-19_002613051.png" alt=""><figcaption><p>Después de completar tu cuenta de email, hacer click en esta casilla para continuar.</p></figcaption></figure>

<figure><img src=".gitbook/assets/imagen_2023-09-19_002826774.png" alt=""><figcaption><p>Lo siguiente es crear una contraseña, una vez escrito volver a hacer click en continuar.</p></figcaption></figure>

{% hint style="info" %}
Seleccionar una contraseña óptima, es decir segura, para ello se recomienda una contraseña con 15 caracteres, o de 8 caracteres combinando letras mayúsculas y minúsculas, y también números.
{% endhint %}

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 003009.png" alt=""><figcaption><p>La contraseña no debe ser muy corta, se sugiere que contenga 15 caracteres, o 8 caracteres con letras mayúsculas y minúsculas, e incluir números.</p></figcaption></figure>

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 003039.png" alt=""><figcaption><p>En este caso la contraseña es medianamente segura, la plataforma hace hincapié en la sugerencia de la cantidad de caracteres.</p></figcaption></figure>

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 003136.png" alt=""><figcaption><p>Una vez ingresado una contraseña óptima, la casilla se verá verde, ahora podemos continuar.</p></figcaption></figure>

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 005042.png" alt=""><figcaption><p>Por último escribimos nuestro nombre de usuario que utilizaremos en la plataforma.</p></figcaption></figure>

{% hint style="info" %}
Si nuestro nombre de usuario no se encuentra disponible para el registro, la plataforma no nos permitirá continuar.
{% endhint %}

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 011741.png" alt=""><figcaption><p>El nombre de usuario que ingresemos debe estar disponible para poder continuar.</p></figcaption></figure>

{% hint style="info" %}
El nombre de usuario solo puede contener caracteres alfanuméricos o un solo guion, y no puede empezar ni terminar con un guion.
{% endhint %}

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 010747 (1).png" alt=""><figcaption><p>El nombre de usuario solo puede contener caracteres alfanuméricos o un solo guion, y no puede empezar ni terminar con un guion. La traducción de hyphen es guion.</p></figcaption></figure>

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 010940.png" alt=""><figcaption><p>Se admite el uso de guion en el intermedio del nombre de usuario. </p></figcaption></figure>

{% hint style="info" %}
Se puede utilizar números en el nombre de usuario.
{% endhint %}

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 005821.png" alt=""><figcaption><p>La plataforma admite caracteres alfanuméricos para el nombre de usuario.</p></figcaption></figure>

{% hint style="info" %}
También se admite el uso de mayúsculas en el nombre de usuario.
{% endhint %}

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 011444.png" alt=""><figcaption><p>El uso de letras mayúsculas el inicio como al final del nombre de usuario está permitido.</p></figcaption></figure>

<figure><img src=".gitbook/assets/imagen_2023-09-19_002613051.png" alt=""><figcaption><p>Ahora que contamos con un nombre de usuario que cumpla todos los requisitos podemos continuar.</p></figcaption></figure>

Recibir notificaciones en el email.

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 012354.png" alt=""><figcaption><p>Si quieres recibir notificaciones de actualizaciones y anuncios en tu email, deber escribir  ¨y¨ para sí, y ¨n¨ para no.</p></figcaption></figure>

Verificar si no eres un Bot.

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 012821.png" alt=""><figcaption><p>Medida de seguridad diseñada para evitar que programas automatizados (bots) creen cuentas de usuario en un sitio web o aplicación. La verificación de bots ayuda a mantener la integridad de una plataforma en línea al prevenir registros automatizados masivos y actividades maliciosas. Al requerir la participación humana en el proceso de registro, se reduce significativamente la posibilidad de cuentas falsas o actividades fraudulentas</p></figcaption></figure>

{% hint style="info" %}
En este punto la verificación puede no estar interrumpirse por diversos motivos, debemos desactivar el VPN, AdBlock o semejante para poder realizar el rompecabezas de verificación.
{% endhint %}

<div>

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 013024.png" alt=""><figcaption><p>El rompecabezas se ha interrumpido.</p></figcaption></figure>

 

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 013250.png" alt=""><figcaption><p>La causa en este caso el AdBlock, debemos desactivarlo para continuar.</p></figcaption></figure>

</div>

<figure><img src=".gitbook/assets/imagen_2023-09-19_013427945.png" alt=""><figcaption><p>También se interrumpe con una red privada virtual o VPN (Virtual Private Network).</p></figcaption></figure>

<div>

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 014225.png" alt=""><figcaption><p>El rompecabezas de verificación consiste en girar la imagen isométrica de un animal para que coincida con la dirección que señala la mano en la imagen de la izquierda.</p></figcaption></figure>

 

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 014240.png" alt=""><figcaption><p>Una vez completado podemos enviar la respuesta.</p></figcaption></figure>

</div>

Una vez que ingresamos todos nuestros datos solicitados y verificamos que no somos bots procedemos al último paso.

<figure><img src=".gitbook/assets/imagen_2023-09-19_014640169.png" alt=""><figcaption><p>Hacemos click en Create Account ahora que fuimos descartados como bots.</p></figcaption></figure>

Por último debemos ingresar un código numérico el cual es enviado a la cuenta de email ingresada para registrar.

<div>

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 014918.png" alt=""><figcaption><p>Casillas para ingresar el código en GitHub.</p></figcaption></figure>

 

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 015138.png" alt=""><figcaption><p>El código es enviado e la cuenta de email en un correo.</p></figcaption></figure>

</div>

Fin del proceso, la cuenta ha sido creada.

<figure><img src=".gitbook/assets/Captura de pantalla 2023-09-19 015344 (1).png" alt=""><figcaption><p>Finalmente logramos crear nuestra cuenta de GitHub, ¡Enhorabuena!</p></figcaption></figure>
